Future<void> doIt()

Executes the asynchronous operation, updates the loading state, and notifies listeners about the result or any error.

Implementation

Future<void> doIt() async {
  _loading = true;
  notifyListeners();
  try {
    final result = await operation();
    _data = result;
    _error = null;
  } catch (e) {
    _error = e.toString();
  } finally {
    _loading = false;
    notifyListeners();
  }
}
